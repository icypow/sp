Клиент отправляет запросы 2 типов:
{Называние очереди}+{Сообщение} - отправка сообщения в очередь (плюс между ними как разделитель имени и сообщения)
{Название очереди} - получение сообщения из очереди
Пример запуска:
python3 udpserver.py 127.0.0.1 9999 client1
python3 udpserver.py 127.0.0.1 9999 client2
python3 udpclient.py 127.0.0.1 9999
